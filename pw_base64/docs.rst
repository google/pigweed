=========
pw_base64
=========
This module provides functions for encoding, decoding, and validating Base64
data as specified by `RFC 3548 <https://tools.ietf.org/html/rfc3548>`_ and
`RFC 4648 <https://tools.ietf.org/html/rfc4648>`_.
